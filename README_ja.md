このコースでは、scikit-learn を使ってデータから予測モデルを構築する方法を学びます。機械学習の基本概念を探索し、scikit-learn を使って教師あり学習と教師なし学習の問題を解く方法を見ていきます。また、モデルの評価方法、パラメータの調整方法、一般的な落とし穴を避ける方法も学びます。実世界のデータセットを使った機械学習の問題の例を通して学んでいきます。
